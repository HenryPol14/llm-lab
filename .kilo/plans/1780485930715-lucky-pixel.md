### Скрипты, требующие проверки/улучшения
Все скрипты имеют разумную степень документации. Некоторые можно улучшить:

1. **03-configure-network.sh** - секция `verify_connectivity` (строки 77-96) - добавить больше пояснений
2. **06-download-cloud-image.sh** - добавить пояснение про формат img
3. **14-deploy-nginx-proxy.sh** - добавить инлайн комментарии для команд `pct exec`

### Решение

Добавить подробные русские инлайн комментарии в следующие функции/блоки:

1. **verify_connectivity()** в `03-configure-network.sh` - комментарии для каждого шага проверки
2. **Download process** в `06-download-cloud-image.sh` - пояснение про временный файл
3. **fix_locale()** и **configure_nginx()** в `14-deploy-nginx-proxy.sh` - комментарии для каждой функции
